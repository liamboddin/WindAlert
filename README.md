# WindAlert

## Starting it locally

### Backend

#### Environment variables

```
PG_HOST=localhost;
PG_PASSWORD=password;
PG_PORT=15432;
PG_USER=admin;
```

#### CLI options

```
--app.ui=./frontend/dist
```

### Frontend

#### Build for 8080

```
npm run build
```

#### Dev with hot reloading

```
npm run dev
```
